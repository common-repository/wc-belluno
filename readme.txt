== Instalação ==

Instale o plugin de duas maneiras diferentes: automaticamente, na seção “Plugins” do WordPress, ou manualmente, baixando e copiando os arquivos do plugin para o seu diretório.

= Instalação automática pelo administrador do WordPress =

1. Acesse "Plugins" no menu lateral de navegação do administrador do WordPress.
2. Uma vez dentro dos plugins, clique em 'Adicionar novo' e pesquise 'Gateway de pagamento Belluno' na lista de plugins do WordPress
3. Clique em "Instalar".

Pronto! Ele estará na seção "Plugins instalados" e a partir daí, você poderá ativá-lo.

= Instalação Manual =
1. Baixe o arquivo zip do plugin no <a href="https://wordpress.org/plugins/wc-belluno/">diretório WordPress</a>.
2. Descompacte a pasta e renomeie-a para "**wc-belluno**".
3. Copie a pasta "**wc-belluno**" para o diretório do WordPress, dentro da pasta "**Plugins**".

**Pronto!**

Instalar este plugin não afeta a velocidade de sua loja!

Se você instalou corretamente, você verá o plugin da Belluno em sua lista de "Plugins Instalados" do WordPress. Habilite-o e prossiga para a integração e configuração de sua conta na Belluno. 

== Integração Belluno ==
1. Crie uma <a href="https://belluno.digital/comece-agora/">conta digital Belluno</a> se você ainda não tem uma. É grátis e leva apenas alguns segundos!
2. Obtenha o seu <a href="http://belluno.tawk.help/article/como-gerar-um-token-de-integracao">token de integração</a>, que é a chave que identifica sua loja de forma única dentro da plataforma.
3. Defina as preferências de pagamento do checkout e faça outras configurações avançadas para alterar as opções padrões.
4. Comece a vender online e receber os pagamentos em sua conta.

== Configuração ==

Siga estas instruções de cinco etapas e prepare tudo para receber pagamentos:

1. Adicione suas **credenciais** para testar a loja e cobrar com sua conta da Belluno.
4. Configure **preferências de pagamento**.
5. Acesse **as configurações** apenas quando desejar alterar as opções padrões.

Confira nossa <a href="http://belluno.tawk.help/article/como-configurar-o-plugin-woocommerce-na-sua-loja-virtual">documentação oficial</a> para mais informações sobre como configurar o plugin.
